metabrainz.el is a collection of Emacs Lisp packages that connect Emacs to the [MusicBrainz](https://musicbrainz.org) open music encyclopedia and the [ListenBrainz](https://listenbrainz.org) listen-tracking service. Use it interactively in `org-mode` to explore music metadata, or integrate it programmatically into music players and workflows to submit listening data.

## Get started in minutes

<Steps>
  <Step title="Install the packages">
    Clone the repository and add it to your Emacs `load-path`, or install via straight.el or use-package.

    ```emacs-lisp
    (add-to-list 'load-path "/path/to/metabrainz")
    (require 'musicbrainz)
    (require 'listenbrainz)
    ```
  </Step>
  <Step title="Configure your ListenBrainz token">
    For ListenBrainz features that require authentication, set your API token. Find your token at [listenbrainz.org/profile](https://listenbrainz.org/profile/).

    ```emacs-lisp
    (setq listenbrainz-api-token "your-token-here")
    ```
  </Step>
  <Step title="Search for music metadata">
    Search MusicBrainz for an artist and get their MBID:

    ```emacs-lisp
    (musicbrainz-artist-to-mbid "Autechre")
    ;; => "410c9baf-5469-44f6-9852-826524b80c61"
    ```
  </Step>
  <Step title="Track what you're listening to">
    Submit a listen to ListenBrainz from Emacs:

    ```emacs-lisp
    (listenbrainz-submit-playing-now "Autechre" "Gantz Graf")
    ```
  </Step>
</Steps>
